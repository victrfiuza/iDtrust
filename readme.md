## 2 - De acordo com o algoritmo a seguir, escolha uma das alternativas abaixo.
```python
contador :=  3
soma  := 57
for (contador <= 10 ) {
	if (contador < 5 || contador == 8 ) {
		soma := soma - contador
    } else {
            soma = soma + contador
    }
	contador++
} 
print("O valor da soma é " + soma)
```
